# Календарная функциональность Forex Bot

## Обзор

В бот добавлена новая функциональность календаря, которая позволяет пользователям выбирать дату для поиска новостей с помощью интерактивного календарного интерфейса.

## Новые возможности

### 1. Команды календаря

- `/calendar` - Открывает интерактивный календарь для выбора даты
- `/choose_date` - Альтернативная команда для открытия календаря

### 2. Интерактивный календарь

Календарь предоставляет следующие возможности:

- **Навигация по месяцам**: Кнопки "◀" и "▶" для перехода между месяцами
- **Быстрый выбор сегодняшней даты**: Кнопка "Сегодня"
- **Выбор конкретной даты**: Клик по любому дню месяца
- **Визуальные индикаторы**: Сегодняшняя дата выделена квадратными скобками [день]
- **Ограничения**: Прошедшие даты неактивны для выбора
- **Закрытие календаря**: Кнопка "❌ Закрыть"

### 3. Выбор уровня важности

После выбора даты пользователю предлагается выбрать уровень важности новостей:

- 🔴 **HIGH** - Высокая важность
- 🟡 **MEDIUM** - Средняя важность  
- 🟢 **LOW** - Низкая важность
- 📊 **ALL** - Все уровни важности

## Использование

### Для пользователей

1. Отправьте команду `/calendar` или `/choose_date`
2. Используйте кнопки навигации для перехода к нужному месяцу
3. Нажмите на день для выбора даты
4. Выберите уровень важности новостей
5. Получите новости за выбранную дату

### Для разработчиков

```python
from app.bot.utils.calendar import create_calendar_markup, process_calendar_callback

# Создание календаря
markup = create_calendar_markup(year=2025, month=6)

# Обработка callback
action, selected_date, navigation_data = process_calendar_callback(callback_data)
```

## Локализация

Календарь поддерживает русский язык:
- Названия месяцев на русском
- Сокращенные названия дней недели
- Русские сообщения и кнопки

## Callback Data Format

Календарь использует следующие форматы callback data:

- `calendar_select_YYYY_MM_DD` - выбор даты
- `calendar_prev_YYYY_MM` - предыдущий месяц
- `calendar_next_YYYY_MM` - следующий месяц
- `calendar_today` - выбор сегодняшней даты
- `calendar_close` - закрытие календаря
- `calendar_ignore` - игнорируемые кнопки
- `date_impact_LEVEL_YYYY-MM-DD` - выбор уровня важности для даты

## Ограничения

1. Календарь не позволяет выбирать прошедшие даты
2. Максимальная дата ограничена разумными пределами
3. Календарь оптимизирован для мобильных устройств (ширина 7 кнопок)
